# Retail-Sales-Dashboard-Analysis
"An interactive Excel Business Intelligence dashboard using Power Pivot, DAX, and Data Modeling"
Project Overview
This project transforms raw retail transaction data into an interactive Business Intelligence (BI) dashboard using Microsoft Excel. It demonstrates a full data lifecycle: from cleaning messy data to building a relational data model.
